DeepCatch X (X-ray) [details]
  • Showcased DeepCatch X at NVIDIA GTC 2025, demonstrating novel generative AI applications in medical imaging to a global audience.
  • Pioneered a Conditional Diffusion model for X-ray-to-Aorta extraction achieving superior fidelity to GANs; proposed LDM-based volumetric reconstruction; accelerated training via DDP and Gradient Accumulation.
  • Engineered a scalable inference pipeline using Ray Distributed across multi-node workstations; achieved 6x batch throughput and 2.8x faster per-case inference via adaptive DAG scheduling and hardware-aware optimizations (TensorRT, ONNX, FP16).
  • Reduced codebase size by 76% via refactoring and built a Nuitka compilation pipeline to deploy models as standalone executables.
  • Implemented a BioAge prediction module by porting research-grade R algorithms to a C++ production environment.
  • Directed the end-to-end acquisition of massive-scale medical datasets (100TB+) for FDA/MFDS regulatory submissions.
3D FaceLandmark and AR Registration for EVD Surgical Navigation [details]

LLM-based De-Identification System and Translation with Agentic Systems [details]
