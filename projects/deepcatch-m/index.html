Overview

DeepCatch M is an AI-powered MRI analysis platform for automated segmentation of multiple organ structures. The project focuses on accurate quantification of anatomical features from MRI scans, supporting clinical decision-making and research applications.

PyTorch MLFlow DVC Medical Imaging Segmentation

Multi-organ MRI Segmentation Models

Problem

Manual segmentation of organs in MRI scans is time-consuming and subject to inter-observer variability. Automated segmentation enables consistent, reproducible quantitative analysis at scale.

Approach

Trained and evaluated deep learning models for multi-organ segmentation on MRI, targeting specific anatomical structures including subcutaneous fat (SF) and abdominal visceral fat (AVF).

Technical Details

  • Developed segmentation models for multiple organ targets
  • Implemented comprehensive evaluation with quantitative metrics (Dice, IoU, Hausdorff distance)
  • Created validation pipelines for cross-dataset generalization testing
  • Optimized model architectures for inference efficiency

Target Structures

  • Subcutaneous Fat (SF)
  • Abdominal Visceral Fat (AVF)
  • Additional organ structures

MLOps Pipeline Template

Problem

Reproducible machine learning experiments require systematic tracking of experiments, model versions, and artifacts. Ad-hoc approaches lead to difficulty in comparing results and deploying the best models.

Approach

Built a reusable MLOps pipeline template that standardizes experiment tracking and model weight management across the team's projects.

Pipeline Components

  • Experiment Tracking: MLFlow integration for logging metrics, parameters, and artifacts
  • Model Registry: Centralized model version management with deployment status tracking
  • Data Versioning: DVC integration for reproducible dataset management
  • Config Management: Hydra-based configuration for experiment reproducibility
  • Automated Reports: Generation of comparison reports across experiment runs

Impact
